# Ametzo Academy

**Ametzo Academy** is a **learning platform and admin dashboard** developed by **Ametzo Technologies**.

It is built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**, providing a modern, scalable, and high-performance foundation for managing **online courses, students, instructors, and learning operations**.

The dashboard is designed to support **course management, enrollments, users, analytics, content administration, and academic workflows** in a clean and extensible UI.

---

## Overview

Ametzo Academy is a feature-rich **Learning Management System (LMS) admin panel** designed for educational platforms.
It provides reusable components, responsive layouts, and a modern developer experience for managing digital learning environments.

---

## Tech Stack

* Next.js 16
* React 19
* TypeScript
* Tailwind CSS v4

---

## Features

* Responsive admin layout with sidebar navigation
* Course management dashboard
* Student and instructor management
* Enrollment and progress tracking
* Data tables and analytics charts
* Forms, modals, alerts, dropdowns, and buttons
* Authentication-ready UI
* Dark mode support
* Modular and reusable components

## Installation

### Prerequisites

* Node.js 18.x or later (Node.js 20.x recommended)

---

### Setup

Clone the repository:

```bash
git clone <your-repository-url>
cd ametzo-academy
```

Install dependencies:

```bash
npm install
# or
yarn install
```

Start the development server:

```bash
npm run dev
# or
yarn dev
```

The application will be available at:

```
http://localhost:3000
```
